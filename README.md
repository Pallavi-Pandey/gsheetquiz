# Gsheet Quiz App

A lightweight quiz application that utilizes Google Sheets as its primary database for storing questions and user responses.

## Key Features
- **Dynamic Content**: Easily update quiz questions through a Google Sheet.
- **Data Persistence**: Responses are recorded directly back to the spreadsheet.
- **Seamless Integration**: Uses Google Apps Script or a simple REST API to connect the sheet.

## Getting Started
Configure your Google Sheet ID and authorized API key in the project settings.
